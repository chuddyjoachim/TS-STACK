## Prerequisite

- Node.js installed
- Yarn installed
- Basic javascript and typescript knowledge
- Express.js
- React.js
- Basic undestanding of graphql and how it works
- Apollo-graphql
- Graphql-code-gen
- Next.js
- Use of databases
- Use of orm e.g micro-orm or typeorm
